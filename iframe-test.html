Dynamic Content Update Example

The portfolio component supports dynamic content updates via JavaScript messaging. Click the button below to see how external sites can customize the content.

// Example of updating portfolio content from parent page const iframe = document.getElementById('portfolio-iframe'); iframe.contentWindow.postMessage({ type: 'portfolioUpdate', content: { title: 'CUSTOM.EXE', about: '<p>Custom about content...</p>', projects: [ { title: 'Custom Project', status: 'ACTIVE', description: 'A customized project description.', technologies: ['React', 'Node.js', 'MongoDB'] } ] } }, '*');
